# Travel Info Explorer

## Overview

A full-stack travel information platform that helps users discover destinations worldwide.

## Features

- User Authentication
- Destination Search
- Travel History
- Admin Dashboard
- Responsive Design

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Bootstrap
- JavaScript
